# Data Analytics Internship Projects — DecodeLabs

## About
This repository contains all projects completed during my Data Analytics Internship at 
DecodeLabs (Batch 2026). The internship follows a structured, end-to-end analyst workflow — 
starting with raw, messy data and progressing through cleaning, exploratory analysis, and 
SQL-based querying to extract real business insights.

Each project builds on the previous one, using the same underlying e-commerce orders 
dataset (1,200 records) to simulate a realistic analyst workflow from raw data to 
actionable insight.

## Projects

### 📁 [Project 1: Data Cleaning & Preparation](./Project1_Data_Cleaning)
Cleaned a raw e-commerce orders dataset by handling missing values, duplicate records, 
and inconsistent formatting using Excel Power Query. Every change was documented in a 
dedicated Change Log for full transparency and auditability.

**Tools:** Excel (Power Query)

---

### 📁 [Project 2: Exploratory Data Analysis](./Project2_EDA)
Performed a deep-dive statistical and visual analysis of the cleaned dataset using Python — 
uncovering distribution patterns, detecting outliers via the IQR method, tracking monthly 
revenue trends, and quantifying the relationship between order quantity and value.

**Tools:** Python (Pandas, Seaborn, Matplotlib), Jupyter Notebook

---

### 📁 Project 3: SQL Data Analysis *(Coming soon)*
Will use SQL queries to filter, group, and aggregate the cleaned dataset — bridging the 
gap between raw data and specific, actionable business answers.

**Tools:** SQL
