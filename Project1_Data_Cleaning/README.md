# Project 1: Data Cleaning & Preparation

## Objective
Clean a raw e-commerce orders dataset (1200 records) by handling missing values, 
duplicates, and inconsistent formatting to prepare it for analysis.

## What Was Done
- Removed duplicate order records
- Standardized inconsistent city/text formatting
- Fixed mixed date formats to a consistent format
- Rounded UnitPrice and TotalPrice to 2 decimal places
- Documented all changes in a Change Log

## Tools Used
Excel (Power Query)

## Files
- `Project1_Data_Cleaning_SujalSinghNegi.xlsx` — Cleaned dataset with Change Log sheet

## Key Outcome
Transformed a raw dataset into a clean, analysis-ready format with full documentation 
of every change made.
